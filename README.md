## 💻 Системные требования
- **ОС:** Windows 10 / 11 (x64)
- **Разрешение экрана:** 1280x1024 и выше
- **Место на диске:** 50 МБ + место под сборки
- **Поддерживаемые форматы:** `.bin`, `.gen`, `.md`, `.smd`

✅ Программа корректно работает при DPI-масштабировании 100%, 125%, 150% и 175%.

## 🗃 База данных (`mrzr_base.dat`)
Файл содержит информацию о **всех официальных играх SEGA Mega Drive / Genesis**, выпущенных на картриджах в **1988–1998 гг.**:

- ✅ Все региональные версии (USA, JAP, EUR + эксклюзивы других стран)
- ✅ Все известные ревизии
- ✅ Уникальная жанровая классификация (до 4 уровней)
- ✅ Типы мультиплеера (Co-op, PvP, Turn-base, Hot-seat)

**Всего:** 877 уникальных игр и 1305 вариантов с учётом версий и ревизий.

> 💡 Для просмотра полной структуры жанров в главном окне нажмите на иконку **📊**.

## ⚙️ Файл настроек (`mrzr_settings.json`)
Создаётся и обновляется автоматически при закрытии программы.

| Параметр | Описание |
|----------|----------|
| `last_base_folder` | Путь к папке с исходными ROM'ами |
| `auto_open_folder` | Открывать папку сборки после создания (`true`/`false`) |
| `last_mode` | Последний тип сборки (`1`=FULL, `2`=1G1R, `3`=2P) |
| `last_region` | Последний регион (`1`=USA, `2`=JAP, `3`=EUR) |
| `window_width` | Ширина окна (≥1024) |
| `window_height` | Высота окна (≥768) |
| `window_x` | Позиция окна по X |
| `window_y` | Позиция окна по Y |
| `show_tooltips` | Показывать всплывающие подсказки (`true`/`false`) |

> ⚠️ Параметр `show_tooltips` изменяется только вручную (при закрытой программе), остальные — автоматически.

## 🚀 Возможности

### 📌 Варианты сборки
- **Три режима:** `[FULL]`, `[1G1R]`, `[2P]`
- **Выбор основного региона:** USA (NTSC-U), JAP (NTSC-J), EUR (PAL)
- **Три типа сортировки:** по жанрам, по мультиплееру, без сортировки

### 🔧 Основные функции
- 🔍 Сканирование папки — находит ROM'ы в указанной и всех вложенных папках
- 🧠 Умная идентификация — определяет ROM'ы по содержимому, а не по имени файла
- 📈 Приоритет ревизий — выбирает более позднюю/стабильную ревизию игры
- 📂 Формирование сборки — создаёт новую структуру методом копирования

### 🛡 Надёжность и контроль
- 💾 Сохранение настроек — запоминает последние выбранные параметры
- ⏹ Прерывание сборки — остановка процесса с удалением незавершённой папки
- 🔔 Уведомление об обновлении — при запуске сообщает в лог о новой версии

### 🎨 Интерфейс и удобство
- 💬 Всплывающие подсказки — подробные пояснения вариантов сборки
- 📂 Автооткрытие папки — готовая сборка открывается автоматически
- 🌳 Структура жанров — просмотр дерева жанров прямо из базы (иконка **📊**)
- 📋 Копирование лога и дерева жанров — частичное или полное через `Ctrl+C` / ПКМ
- 📱 DPI-коррекция — поддержка масштабирования экрана 100%, 125%, 150%, 175%

## 🎮 Использование
Создание сборки за три простых шага:

1️⃣ **Укажите папку с ROM'ами** (клик по синей панели внизу окна)  
2️⃣ **Выберите параметры сборки** (см. подробные подсказки при наведении курсора)  
3️⃣ **Нажмите «СОЗДАТЬ СБОРКУ»** (готовая сборка появится в папке `ROMSETS`)

### Формат имени папки сборки:
   СБОРКА [РЕГИОН] [СОРТИРОВКА] [КОЛ-ВО ФАЙЛОВ] [ДАТА_ВРЕМЯ]

**Примеры:**
- `FULL [1652] [2026.03.03_00-04-27]`
- `1G1R [PRIO-JP] [GENRE] [877] [2026.03.03_00-04-52]`
- `2P [PRIO-EU] [MP-TYPE] [250] [2026.03.03_00-05-26]`
